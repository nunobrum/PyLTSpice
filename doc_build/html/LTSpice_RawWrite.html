LTSpice_RawWrite.py¶

This module generates RAW Files from user data. It can be used to combine RAW files generated by different Simulation Runs

class PyLTSpice.LTSpice_RawWrite.LTSpiceRawWrite(plot_name='', fastacces=True)[source]¶

Bases: object

This class represents the RAW data file being generated. Contrary to the the LTSpiceRawRead this class doesn’t support stepped data.

add_trace(trace: PyLTSpice.LTSpice_RawWrite.Trace)[source]¶

Adds a trace to the RAW file. The trace needs to have the same size as trace 0 (‘time’, ‘frequency’, etc..) The first trace added defines the X-Axis and therefore the type of RAW file being generated. If no plot name was defined, it will automatically assign a name. :param trace: Needs to be of the :type trace: :return: Nothing :rtype: None
